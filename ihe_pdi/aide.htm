So arbeiten Sie mit diesem Datenträger

 

Dieser Datenträger enthält Bilder im Format JPEG.

JPEG-Bilder

Diese Bilder haben feste Windowing-Einstellungen.
Die Bildanzeige kann nicht geändert werden (keine Zoomfunktionen, keine Dichte- und Abstandsmessungen).
 Sind animierte Bilder auf dem Datenträger vorhanden, wird jeweils nur ein Standbild aus der ersten Sequenzhälfte angezeigt.


Die auf diesem Datenträger gespeicherten Bilder sind nicht bearbeitbar.
Sie können auf jedem beliebigen Computer mit einem Webbrowser angezeigt werden.

Wenn Sie auf das Symbol oben klicken, wird von jeder auf dem Datenträger gebrannten Studie das erste Bild jeder Serie angezeigt.
Wenn Sie auf eines der Bilder klicken, wird eine Seite
Wenn Sie auf ein Bild dieser Serie klicken, wird das Bild vergrößert dargestellt und eine Navigationsleiste eingeblendet, mit der Sie durch die weiteren Bilder der Serie blättern können.
