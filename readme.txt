Dieser Datenträger kann mittels eines Webbrowers auf Macintosh-, Unix- und Linux-Systemen gelesen werden.


HINWEISE FÜR MACINTOSH-BENUTZER


Um diesen Datenträger auf einem Macintosh-System zu lesen, doppelklicken Sie auf die Datei Index.htm. Ihr Standard-Webbrowser wird automatisch gestartet. Ist dies nicht der Fall, starten Sie Ihren bevorzugten Webbrowser und klicken Sie im Menü Datei auf Datei öffnen. Gehen Sie zum Stammverzeichnis des Datenträgers und wählen Sie dort die Datei Index.htm.


HINWEISE FÜR WINDOWS-BENUTZER

Wenn auf Ihrem System die automatische Wiedergabe aktiviert ist, wird der Datenträger automatisch geöffnet. Folgen Sie der Anleitung am Bildschirm. Als Alternative können Sie auch den Windows Explorer öffnen, mit der rechten Maustaste auf das CD-/DVD-Laufwerk klicken und dann die automatische Wiedergabe aktivieren.
